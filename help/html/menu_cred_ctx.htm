Credential Display Context Menu

Click an item on the menu to go to the description of the action, or choose from the list below. You can activate the credential display context menu by right-clicking on an identity or credential in the credentials display.

Actions which have an associated hot key show this hot key to the right of the action. You can use the hot key to trigger the action without invoking the menu.

  • Properties: Displays the property sheet for the selected identity or credential or credentials type.

  • Show identity options: Invokes the per-identity configuration panel for the selected identity.

  • Set as default: Sets the selected identity as the default.

  • Obtain new credentials ...: Invokes the New credentials wizard. If an identity is currently selected, that identity will be pre-selected as the primary identity for the wizard.

  • Renew credentials: Attempts to renew the selected credentials or identity. See the Renew credentials action for more information.

  • Destroy credentials: Attempts to destroy the selected credentials. See the Destroy credentials action for more information.
