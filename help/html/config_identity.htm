Per identity configuration

You can access the per-identity configuration panel for a specific identity by selecting the identity name from the list of configuration panels in the configuration dialog.

Alternatively, you can right-click on an identity in the credentials display and select the Show identity options menu item to bring up the configuration dialog for that identity.

Many of the credentials acquisition settings that are available in this dialog are also exposed in the advanced view of the New credentials wizard.

These panels are similar to the Identities configuration panel, but they change per-identity settings. Changes you make in these panels will override the defaults set in the Identities panel.

There are several standard panels that form the per-identity configuration dialog:

  • General configuration
  • Kerberos v5 configuration
  • Kerberos v4 configuration
  • KeyStore configuration
